Autosense Smartroom

A tiny smart system making your room intelligent ✨

This project automates your room using ESP32 + Firebase, enabling real-time motion alerts, energy-efficient control of lights & fans, and smart power-saving features.

Go to Dashboard

Project Overview

The Autosense Smartroom combines ESP32 + Firebase to automate room lighting, fan control, temperature monitoring, and energy management with intelligent real-time responses.

Key Features

Real-Time Alerts

Instant beep sound when motion is detected.

Night Mode

Automatically disables devices after 11 PM.

Power Saving Mode

One-tap shutdown of lights and fan.

Usage Label

Shows Today’s Usage: Light / Medium / High.

Daily Energy Report

Shows exactly how long each device ran today.

Cloud Sync

Fully synced with Firebase in real time.
